# Aula 01 - Python do Zero

## Contexto

Primeiro contato com Python: executar scripts, ler entradas do usuario e fazer operacoes basicas.

## Arquivos da pasta

- `exemplo_001.py`: primeiro programa com `print`.
- `desafio.py` e `desafio_4.py`: leitura de nome e uso de `len()`.
- `desafio_2.py` e `desafio_5.py`: soma de dois valores com conversao de tipo.
- `desafio_3.py`: calculo de bonus com variaveis e `f-string`.
- `main.py`: arquivo reservado (sem conteudo).

## Conceitos-chave praticados

- Entrada e saida: `input()` e `print()`.
- Tipos basicos: `str`, `int`, `float`.
- Conversao de tipos.
- Operadores aritmeticos (principalmente soma).
- Tamanho de texto com `len()`.
- Formatacao de mensagem com `f-string`.

## Objetivo de estudo

Consolidar a base de programacao em Python antes de avancar para validacao e tratamento de erros.
