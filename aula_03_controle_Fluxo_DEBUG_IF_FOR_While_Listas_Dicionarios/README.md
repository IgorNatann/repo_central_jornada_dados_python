# Aula 03 - Controle de Fluxo, Listas e Dicionarios

## Contexto

Aplicacao de logica de programacao em cenarios de dados: validacao, classificacao, filtros, agregacao e repeticao.

## Arquivos da pasta

- `exercicios.py`: conjunto de exercicios com `if`, `for`, `while`, listas, dicionarios e funcoes simples.

## Conceitos-chave praticados

- Regras com operadores logicos (`and`, `or`).
- Classificacao por faixa com `if/elif/else`.
- Leitura e validacao de campos em dicionarios.
- Contagem de ocorrencias com acumulador em dicionario.
- Agregacao por categoria (somatorio incremental).
- Filtragem de registros completos e incompletos.
- Extracao de subconjuntos (ex: numeros pares).
- Normalizacao de valores para escala 0 a 1.
- Loops `for` e `while`, com condicao de parada (`break`).
- Simulacoes de fluxo real: paginacao de API e tentativas de conexao.

## Objetivo de estudo

Ganhar fluidez para resolver problemas de dados com estruturas de controle e estruturas de dados nativas do Python.
