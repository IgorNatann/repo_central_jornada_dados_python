# Aula 04 - Tipos Complexos e Type Hints

## Contexto

Introducao ao uso de type hints para deixar o codigo mais legivel, previsivel e facil de manter.

## Arquivos da pasta

- `type_hint.py`: explicacao introdutoria e exemplos de anotacao de tipos.
- `exercicio_type_hint.py`: validacao de entrada com variaveis anotadas.
- `exercicios_list_dict.py`: pratica com listas e dicionarios tipados.

## Conceitos-chave praticados

- Type hints em variaveis: `int`, `str`, `float`, `bool`.
- Type hints em colecoes: `list[int]`, `list[str]`, `dict[str, int]`, `dict[str, float]`.
- Validacao de entrada com `while` + `try/except`.
- Operacoes de lista (`append`, `remove`).
- Uso de dicionario para contagem e busca de precos.
- Calculo acumulado com validacao de chave ausente.

## Objetivo de estudo

Dominar anotacao de tipos e reforcar boas praticas em estruturas de dados usadas no dia a dia de dados.
